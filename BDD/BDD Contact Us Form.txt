Feature: Contact Us Form Functionality
  As a website user
  I want to interact with the Contact Us page
  So that I can communicate with the business and get support

  Background:
    Given I am on the Contact Us page

  @TC-CONTACT-001
  Scenario: Verify successful form submission with valid data
    When I enter name "Ahmed Qatar"
    And I enter email "ahmedqattar@outlook.com"
    And I enter subject "Test Subject"
    And I type message "This is a test"
    And I upload file "test.txt"
    And I click Submit button
    Then I should see success message "Success! Your details have been submitted successfully."

  @TC-CONTACT-002
  Scenario: Verify form validation for empty email field
    When I fill all fields except email
    And I leave email blank
    And I click Submit button
    Then I should see error message near email field
    And the form should not submit

  @TC-CONTACT-003
  Scenario: Verify file upload functionality with image
    When I enter name "Ahmed Qatar"
    And I enter email "ahmedqattar@outlook.com"
    And I enter subject "Test Subject"
    And I type message "Test message"
    And I upload image file "photo.jpg"
    And I click Submit button
    Then the file should be attached
    And the form should submit successfully with image included

  @TC-CONTACT-004
  Scenario: Verify form validation when only email field is filled
    When I leave name field blank
    And I enter email "ahmedqattar@outlook.com"
    And I leave subject field blank
    And I leave message field blank
    And I do not upload any file
    And I click Submit button
    Then the form should not submit
    And I should see validation errors for mandatory fields

  @TC-CONTACT-006
  Scenario: Verify form validation with invalid email format
    When I enter name "Test User"

    And I type message "Test message"
    And I click Submit button
    Then the form should not submit
    And I should see error message for invalid email format

  @TC-CONTACT-007
  Scenario: Verify feedback email link functionality
    When I locate feedback email address
    And I click on email link "feedback@automationexercise.com"

  @TC-CONTACT-008
  Scenario: Verify home navigation after successful submission

    When I click the home link

    And I upload Excel file "document.xlsx"

    And I click Submit button

    When I fill all required fields with valid data

  Scenario: Verify file size limit validation
